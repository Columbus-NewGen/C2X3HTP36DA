## 1) /dashboard — แดชบอร์ด

**สำหรับ**: Owner/Manager  
**หน้าที่**: ดูสถานะยิมแบบรวม (Home Page)

**แสดงข้อมูล:**

- 📊 จำนวนสมาชิกทั้งหมด
- 👥 ผู้ใช้งานวันนี้
- 🏋️ จำนวนการใช้เครื่องต่อวัน
- ⚠️ สถานะเครื่อง (ว่าง/ซ่อมบำรุง)
- 📈 กราฟสถิติพื้นฐาน

## 2) /floorplan — รายการผังพื้นที่

**สำหรับ**: Owner/Manager  
**หน้าที่**: จัดการผังพื้นที่ของแต่ละชั้น

**สามารถทำได้:**

- 📋 ดูรายการผังพื้นที่ (ชั้น 1, 2, 3 ฯลฯ)
- 📊 เห็นสถานะการใช้งานเครื่องแบบรวม
- ✏️ ปุ่มแก้ไข (Edit)
- 👁️ ปุ่มดูตัวอย่าง (Preview)

## 3) /floorplan/editor/:id — แก้ไขผังพื้นที่ ⭐

**สำหรับ**: Owner/Manager  
**หน้าที่**: ลากและวางเครื่องเล่นในแผนพื้นที่ (Core Feature)

**ฟีเจอร์หลัก:**

- 🖱️ ลากวาง Drag & Drop เครื่องเล่น
- 🏷️ ตั้งประเภทเครื่อง เลขหมายเครื่อง
- 🔄 สลับสถานะ Active / Maintenance
- 🔗 เชื่อมต่อกับข้อมูลเครื่องในระบบ
- 📏 Snap-to-Grid สำหรับจัดวางง่าย
- 💾 บันทึกตำแหน่งแบบ Real-time

## 4) /machines — รายการเครื่องเล่น

**สำหรับ**: Owner/Manager  
**หน้าที่**: ดูและจัดการเครื่องเล่นทั้งหมด

**แสดงข้อมูล:**

- ชื่อเครื่อง (เช่น Leg Press #1)
- กล้ามเนื้อที่ใช้
- สถานะ (Active / Maintenance)
- จำนวนครั้งถูกใช้งาน
- ปุ่มดูรายละเอียด

## 5) /machines/:id — รายละเอียดเครื่องเล่น

**สำหรับ**: Owner/Manager/Member  
**หน้าที่**: ดูข้อมูลเฉพาะของเครื่องแต่ละเครื่อง

**แสดงข้อมูล:**

- 📷 รูปเครื่อง
- 📖 วิธีใช้งาน / คำแนะนำ
- 🎥 วิดีโอสาธิต
- 🔧 ข้อมูล Maintenance
- 📊 ประวัติการใช้งาน
- 🔄 เครื่องทดแทนที่สามารถใช้ได้

## 6) /announcements — ประกาศข่าวสาร

**สำหรับ**: Owner/Manager  
**หน้าที่**: ทำประกาศและส่งข้อมูลให้สมาชิก

**ฟีเจอร์:**

- 📢 เขียนและปรับแบนเนอร์ประกาศ
- ⏰ ตั้งเวลาหมดอายุข่าว
- 🔔 ส่ง Notification ให้สมาชิก
- ⚠️ ข่าวซ่อมบำรุงเครื่อง

## 7) /programs — โปรแกรมฝึก

**สำหรับ**: Trainer/Owner  
**หน้าที่**: จัดการโปรแกรมฝึกที่สำเร็จแล้ว

**ฟีเจอร์:**

- 📋 สร้างโปรแกรมฝึกแบบ Preset
- 📊 แยกตามระดับ: Beginner, Intermediate, Advanced
- 📥📤 Import/Export โปรแกรม
- 🔗 ผูกโปรแกรมกับเครื่องเล่นที่มีในยิม

## 8) /exercises — ท่าออกกำลังกาย

**สำหรับ**: Trainer/Owner  
**หน้าที่**: จัดการคลังท่าออกกำลังกาย

**แสดงข้อมูล:**

- 📝 รายชื่อท่าทั้งหมด
- 🎥 รูป/วิดีโอประกอบ
- 💪 กล้ามเนื้อที่เกี่ยวข้อง
- 📊 ใช้ในโปรแกรมไหน
- 📋 RPE (ความเหนื่อย) และคำอธิบายสั้น ๆ

## 9) /users — จัดการสมาชิก

**สำหรับ**: Owner/Manager  
**หน้าที่**: ดูและจัดการสมาชิกทั้งหมด

**ฟีเจอร์:**

- 👥 ดูรายชื่อสมาชิก
- 🔐 จัดการ Role (Admin / Manager / Trainer / Member)
- 📈 ดูความคืบหน้าของสมาชิก
- 🚫 Suspend/Block สมาชิก
- 🔑 Reset Password

## 10) /reports — รายงาน

**สำหรับ**: Owner/Manager  
**หน้าที่**: ดูรายงานสถิติการใช้งาน

**รายงานที่แสดง:**

- 📊 ยอดการใช้งานเครื่องแต่ละเครื่อง
- 🔥 Heatmap เครื่องที่ได้รับความนิยม (Peak time)
- 👥 จำนวนสมาชิก Active
- 💰 รายได้/แพ็กเกจ (หากมี)

## 11) /settings — ตั้งค่าระบบ

**สำหรับ**: Owner  
**หน้าที่**: ตั้งค่าข้อมูลยิมและระบบโดยรวม

**ตั้งค่าที่สำคัญ:**

- ⏰ เวลาเปิดปิดยิม
- 🏢 ข้อมูลยิม (ชื่อ ที่อยู่ เบอร์โทร)
- 🔑 API Key / Integration
- 🔐 การตั้งค่า Role & Permission
- 🎮 Gamification (ตัวเลือก)

## 12) /system-logs — ประวัติระบบ

**สำหรับ**: Owner/Admin  
**หน้าที่**: ตรวจสอบความปลอดภัยและติดตามการใช้งาน

**บันทึกที่แสดง:**

- 🔐 Login/Logout ของผู้ใช้
- 📝 การแก้ไขข้อมูลสำคัญ
- 👀 เหมาะสำหรับ Audit Log
- 🛡️ ตรวจสอบความปลอดภัย
